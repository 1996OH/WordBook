```mermaid
sequenceDiagram
    participant ユーザー
    participant フロントエンド
    participant バックエンド
    participant データベース

    ユーザー->>フロントエンド: ログイン画面アクセス
    フロントエンド-->>ユーザー: ログインフォーム表示

    ユーザー->>フロントエンド: ログイン情報入力・送信
    フロントエンド->>バックエンド: ログインAPIリクエスト
    バックエンド->>データベース: ユーザー認証照会
    データベース-->>バックエンド: ユーザー情報返却

    alt ログイン成功
        バックエンド-->>フロントエンド: 認証成功レスポンス
        フロントエンド-->>ユーザー: 単語一覧画面表示
    else ログイン失敗
        バックエンド-->>フロントエンド: 認証失敗
        フロントエンド-->>ユーザー: ログイン画面再表示（エラー付き）
    end

    ユーザー->>フロントエンド: 単語の編集ボタン押下
    フロントエンド->>バックエンド: 編集対象単語の取得リクエスト
    バックエンド->>データベース: 単語詳細取得
    データベース-->>バックエンド: 単語情報返却
    バックエンド-->>フロントエンド: 単語情報返却
    フロントエンド-->>ユーザー: 単語編集画面表示（既存情報を反映）

    ユーザー->>フロントエンド: 名称、正式名称、カテゴリ、意味を編集
    ユーザー->>フロントエンド: 登録ボタン押下

    フロントエンド->>バックエンド: 編集リクエスト（編集後のデータ送信）
    バックエンド->>データベース: 重複チェック（他の単語と重複していないか）
    alt 重複あり
        データベース-->>バックエンド: 重複エントリ存在
        バックエンド-->>フロントエンド: 編集拒否（重複エラー）
        フロントエンド-->>ユーザー: 編集画面再表示（エラー内容表示）
    else 重複なし
        データベース-->>バックエンド: 重複なし
        バックエンド->>データベース: 単語情報を更新
        データベース-->>バックエンド: 更新完了
        バックエンド-->>フロントエンド: 更新成功レスポンス
        フロントエンド-->>ユーザー: 単語一覧画面表示
        Note over ユーザー: 編集後の一覧が表示
    end


 ```
