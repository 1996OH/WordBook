```mermaid
sequenceDiagram
    participant ユーザー
    participant フロントエンド
    participant バックエンド
    participant データベース

    ユーザー->>フロントエンド: ログイン画面アクセス
    フロントエンド-->>ユーザー: ログインフォーム表示

    ユーザー->>フロントエンド: ログイン情報入力・送信
    フロントエンド->>バックエンド: ログインAPIリクエスト
    バックエンド->>データベース: ユーザー認証照会
    データベース-->>バックエンド: ユーザー情報返却

    alt ログイン成功
        バックエンド-->>フロントエンド: 認証成功レスポンス
        フロントエンド-->>ユーザー: 単語一覧画面表示
    else ログイン失敗
        バックエンド-->>フロントエンド: 認証失敗
        フロントエンド-->>ユーザー: ログイン画面再表示（エラー付き）
    end

    ユーザー->>フロントエンド: 新規単語登録ボタン押下
    フロントエンド-->>ユーザー: 単語登録画面表示

    ユーザー->>フロントエンド: 名称、正式名称、カテゴリ、意味を入力
    ユーザー->>フロントエンド: 登録ボタン押下
    フロントエンド->>バックエンド: 単語登録APIリクエスト（入力データ送信）

    バックエンド->>データベース: 単語の重複チェック
    alt 単語すでに存在
        データベース-->>バックエンド: 重複あり
        バックエンド-->>フロントエンド: エラーレスポンス（重複）
        フロントエンド-->>ユーザー: 登録画面再表示（重複エラー表示）
    else 単語未登録
        データベース-->>バックエンド: 重複なし
        バックエンド->>データベース: 新規単語データを登録
        データベース-->>バックエンド: 登録完了
        バックエンド-->>フロントエンド: 登録成功レスポンス
        フロントエンド-->>ユーザー: 単語一覧画面表示
        Note Over ユーザー:単語が追加された状態の一覧が表示
    end

 ```
