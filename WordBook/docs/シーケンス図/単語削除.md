```mermaid
sequenceDiagram
    participant ユーザー
    participant フロントエンド
    participant バックエンド
    participant データベース

    ユーザー->>フロントエンド: ログイン画面アクセス
    フロントエンド-->>ユーザー: ログインフォーム表示

    ユーザー->>フロントエンド: ログイン情報入力・送信
    フロントエンド->>バックエンド: ログインリクエスト
    バックエンド->>データベース: ユーザー認証照会
    データベース-->>バックエンド: ユーザー情報返却

    alt ログイン成功
        バックエンド-->>フロントエンド: 認証成功レスポンス
        フロントエンド-->>ユーザー: 単語一覧画面表示
    else ログイン失敗
        バックエンド-->>フロントエンド: 認証失敗
        フロントエンド-->>ユーザー: ログイン画面再表示（エラー付き）
    end

    ユーザー->>フロントエンド: 単語一覧内の削除ボタン押下

    フロントエンド->>バックエンド: 削除APIリクエスト（単語IDなど含む）
    バックエンド->>データベース: 指定単語を削除
    データベース-->>バックエンド: 削除成功レスポンス
    バックエンド-->>フロントエンド: 成功ステータス返却
    フロントエンド-->>ユーザー: 単語一覧更新（削除反映）

    Note over ユーザー: 削除完了し、画面上から対象行が消える


 ```
